  • Meeting with the Client
    • Get a sense of what the client wants in a website
    • Talk to some customers to get a feel for what they want from the user experience
    • You don't want to start coding right away
    • You want to have some mockups to present the customer
    • Uses balsalmiq for mockup
    • Once the mockups are approved you will generally generate a design in photoshop (glimpse)
    • Use fonts on website that can be found on google fonts
  • Starting coding the website
    • Download bootstrap from the bootstrap getting started page
    • Download jQuery, right click and save link as into your js folder
    • Get the bootsrap starter page from Lecture 25
    • Update the link for jQuery to the correct file/version
    • Get fonts from fonts.google.com
    • Select the fonts you want then copy the html and css code to implement in your index.html and styles.css files respectively
  • Coding Basics of the Navbar Header

    • We coded up the start of the navbar header
