  • Lists
    • html structure that allows you to group content
    • <li> are the only elements allowed directly within an <ul> element
    • You may have unordered list element nested within a list item. The unordered list item must be within the opening and closing tags of the list item to be valid html.
    • If you need an ordered list, use the ordered list element <ol>
    • Ordered lists are numbered rather than bulleted while unordered lists are bulleted.
  • HTML Character Entity References
    • Help avoid rendering issues
    • Can provide characters not available on the keyboard
    • In the below list I am representing the ampersand (&) with the word amp.
    • 3 Characters you should always escape: < (&lt;), > (&gt;), & (&amp;)
    • Others include the copyright symbol © (&copy;)
    • Another escape character commonly used is the non-breaking space character which forces a group of words to not be broken or wrapped around. (ampnbsp;)
    • The nbsp entity reference should be used in place of a space between words with no spaces between the reference and the words.
    • The non-breaking space escape should not be misused to add spaces where margin should be added correctly.
    • Another commonly used character entity reference is that for quotes. This is used for more limited character encoding such as in email. " (ampquot;)
  • Creating Links
    • Links are implemented using an anchor element <a href="foo.html" title="foo">Text displayed<a>
    • Internal links reference a file within a folder locally.
    • If there is no path, html assumes the file resides in the same directory as the parent file.
    • The title attribute is used for screen readers helping the visually impaired get through a webpage.
    • External links are no different but reference an external page usually preceded by http://
    • Anchor elements are both block level and inline elements. You can place a div element within an anchor element to ensure the link shows up on its own line.
    • In an external link you can force a new tab to be opened using the attribute target="_blank"
    • Fragment Identifiers
      • Links within the page
      • href attribute with element id preceded by #. <a href="#id">link text<a> links to <br id="id">
      • name attributes within and anchor element can be used in place of id attributes to create fragment identifiers
      • Example: <h1><a name="id">heading 1</a></h1>
      • This is also convenient as it allows you to share links to a location within a page.
      • These links are also important for spa (single page applications).
  • Displaying Images
    • commenting a line in html with <!-- blah blah -->
    • insert images with an image tag <img src="" width="" height="">
    • width and height are in pixels


    • Always specify width and height because without it may break your layouts
